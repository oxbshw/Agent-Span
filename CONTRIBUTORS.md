# Contributors

AgentSpan is open source and contributions are welcome — bug reports, channels,
SDK ports, docs fixes, and tests all help.

## Maintainers

- AgentSpan Developer — project lead & maintainer

## How to get listed

Open a PR. Once it's merged, add yourself here (alphabetical by handle) in the
same change or a follow-up. Drive-by fixes count.

## A note on tooling

Parts of this codebase were developed with AI assistance (pair-programming with
an LLM) alongside human review. We mention it because it's true and because the
review/testing bar — `fmt`, `clippy -D warnings`, and a green test suite on every
change — is what actually keeps the quality up, regardless of who (or what) drafts
a first pass.

See [CONTRIBUTING.md](CONTRIBUTING.md) for the workflow and coding conventions.
